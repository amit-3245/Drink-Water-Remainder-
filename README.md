# Drink Water Reminder Notification 

This is a simple Python program that sends desktop notifications at regular intervals to remind the user to drink water.

- Infinite loops
- Time delays
- Desktop notifications using external libraries

---

# Features
- Sends a desktop notification reminder
- Prints reminder message in the terminal
- Runs continuously until stopped
- Uses the Plyer library for cross-platform notifications

---

# Technologies Used
- Python 3
- Plyer library
- time module
